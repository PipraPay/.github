## 🚀 What is PipraPay?

PipraPay is an **open-source payment automation system** that lets you:

- Accept payments from any gateway or wallet  
- Automate SMS-based payment verification  
- Build custom payment workflows  
- Connect APIs, banks, and mobile money systems  
- Run everything on your own server  

## ⚡ Key Features

- 🔌 Plugin-based architecture  
- 💳 Multi-gateway support (Stripe, PayPal, bKash, Nagad, UPI, etc.)  
- 📩 SMS payment automation engine  
- 🔗 Webhook & API system  
- 🧩 Custom gateway development support  
- 🏦 Bank & MFS integration layer  
- ⚙️ Fully self-hosted & extensible  


## 🌍 Built for Real-World Payments

PipraPay is designed for regions where:

- No Stripe / PayPal support  
- SMS-based payments are common  
- Manual verification slows down businesses  

We turn those limitations into **programmable automation workflows**.
